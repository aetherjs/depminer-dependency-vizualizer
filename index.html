DepMiner â€” an open source,language-agnostic tool for mining detailed dependencies datafrom source code. DepMiner can be easily integrated into an experimental pipeline to conduct large-scale studies involving intra-project dependencies.

This web app allows you to visualise sample DepMiner output. Select the JSON output file produced by DepMiner and the visualisation shall appear shortly on the graph below.

The simple graph shows structure og the project - each node respresent a file in a project directory and the edge between two nodes represents a dependency that exist between the two. For simplicity's sake dependency direction is not reflected on the graph.

If you don't yet have access to DepMiner's output - you can click Use Sample Output to see a dependency graph for DepMiner itself. The sample output used to build the graph can be obtained by running 

./extract-dependencies src/ src/ .
